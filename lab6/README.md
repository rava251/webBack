# Лабораторная работа №6: Панель администратора

**Автор:** [Раван]
**Стек:** PHP 8, MariaDB, CSS3.

## Описание
В работе реализована административная панель для управления анкетами пользователей.

### Функционал:
* **Авторизация**: Доступ ограничен через HTTP Basic Auth (admin:123456).
* **Статистика**: Вывод количества пользователей по каждому языку программирования.
* **Удаление**: Полное удаление анкеты и связанных данных.
* **Редактирование**: Возможность изменять данные любой существующей анкеты.
