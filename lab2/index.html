Отчет по Заданию №2: Анализ HTTP-запросов

1. Получение главной страницы (HTTP 1.0)

Использование метода GET в протоколе HTTP 1.0. Сервер закрывает соединение после передачи данных.

2. Получение внутренней страницы (HTTP 1.1)

Использование метода GET в протоколе HTTP 1.1. Обязательное использование заголовка Host.

3 и 4. Размер файла и тип ресурса (HEAD)

С помощью метода HEAD определены параметры файла без его загрузки: размер (Content-Length) и MIME-тип (Content-Type).

5. Отправка комментария (POST)

Передача данных на сервер в теле запроса. Проверка работоспособности index.php.

6 и 7. Частичный контент и кодировка

Использование заголовка Range для получения первых 100 байт (статус 206 Partial Content) и определение кодировки UTF-8.

Выполнено с использованием Telnet и Git
